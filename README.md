## About BoardVault

**BoardVault** is a comprehensive mobile application designed to be your ultimate companion for Single Board Computers (SBCs) and Micro Controllers. Whether you're a hobbyist, student, or professional, BoardVault provides essential information, FAQs, and resources to help you master your board development journey.

### Key Features

- **Board Database** - Extensive catalog of SBCs and micro controller information
- **Comprehensive FAQs** - Quick answers to common questions
- **Privacy-First** - Transparent privacy policy and terms

## Known Issues & Troubleshooting

### Common Issues

**Flutter Build Fails**

```bash
flutter clean
flutter pub get
flutter pub upgrade
```

**Backend Connection Errors**

- Verify PostgreSQL is running
- Check DATABASE_URL in `.env`
- Ensure Redis is accessible (if configured)

**API Timeout**

- Check backend is running (`bun run dev`)
- Verify network connectivity
- Check firewall settings
